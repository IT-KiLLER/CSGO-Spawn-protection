# CSGO-Spawn-protection 1.1
Players can not be damaged in a few seconds after spawn.

## Cvars:
  - `sm_spawn_protection_time` `15.0` - *_How long the player will be protected after spawn. 0 = disabled or time 1-180 seconds._*
